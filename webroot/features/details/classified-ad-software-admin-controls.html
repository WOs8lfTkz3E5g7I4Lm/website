Admin Controls

Tools to Manage Users

User Group Pricing
A price plan is attached to each user group that allows you to bill users in that user group a specific way. Each user group can have it's own price plan (way of charging)....
Details
Multi-Admin Users
Create as many admin users within the system as you want. Create admin groups with specific rights and add users to them. You can tailor access and abilities down to a page by page basis as well as view/edit abilities within those pages..
User Export
Exports users from within software to allow import into other software. Allows you to choose all users or by user group to export. Choose which user data fields to export. User Import coming soon...
Admin User Removal
With a few clicks you can remove a user from the system within the admin tool. With that removal all traces attached to the user are removed also. This includes all listings (and their images), saved searches,...etc.
Admin Ability to Manage Feedbacks
You as the admin can change any feedbacks left by any user for any user in the system. You can even remove feedback if you wish. This allows you to remove/edit any inflamatory comments left between disputing clients. Sometimes there are unforeseen issues within a transaction that don't warrant any negative conversation and you have another tool to control that.
Registration Options
Register new users through registration process or within their first listing placement process with little notice. Multiple user field options to hold user data. Multiple user groups can be used to give different prices, privileges, access, and much more..
Bridge Users
Share users between multiple Geodesic Solutions software installations as well as vBulleting forum software. You can also use JFusion plugin to bridge users with Joomla installations...
Suspend Users
This allows you to hold a user in the system yet not allow them access to it. Though you can remove a user from the system easily suspending them locks that email and username in the system so it can't be reused. Many deleted users will simply try to register using the same credentials. This feature stops that..
Login As A User on Client Side
This allows you to do administrative tasks for the client or just see what a client is seeing from their perspective. Great for debugging client account issues or just doing things for them they feel they can't do themselves.
Admin Tool User Account Edit
You as the admin can manually change any aspect of a clients account. Edit any data field attached to their account. You can also change their user group affiliation. Manually verify/unverify their account. Adjust their site account balance. Manually freeze specific site account activity..etc .
User Groups
Allows you to group users for different purposes. You can keep track of ad campaign effectiveness, control access to specific features, charge each user group differently, and more...
Post Login Page Configuration
This feature allows you to set the url the software directs the client to after they log into the system. This allows you to let clients know of specials or information you want them to know when using your site. Or just another option to push ads to the client at a specific time..
Control Client Tool Access by Group
This allows you to control what features users have access to within the client side admin tool using controls in their user group. Certain user groups may not need access to listing management features if they don't have the ability to place listings. Other needs can dictate other custom configurations by user group to suit your needs by user group.
Admin Ability to Manage Feedbacks
You as the admin can change any feedbacks left by any user for any user in the system. You can even remove feedback if you wish. This allows you to remove/edit any inflamatory comments left between disputing clients. Sometimes there are unforeseen issues within a transaction that don't warrant any negative conversation and you have another tool to control that.
Move User Group Into Another User Group
There are many needs for user groups but they sometimes outlast their need. When they do you can move all the users attached to one user group into another user group with just a few clicks in the admin tool. This allows you to quickly consolidate users in multiple users groups into one user group to simplify user management whenever you need.

Tools to Manage Listings And Listing Content

Force Admin Approval
You can force admin approval of many features attached to a listing within your site. You can force admin approval for classifieds and/or auctions, for features your clients attach to their listings, for those using a specific gateway, for those in a specific price plan....any number of ways. When in the system these listings are not made live until you approve them within the admin tool. .
Admin Tool Listing Upgrade
This allows you to immediately add any listing extra free of charge to any listing within the system. You can also remove any listing extra through the same tool. You also have the ability to give more "image/file upload slots" to any listing if you happen to charge to attach images/files within the system.
Admin Tool Listing Edit
You as the admin can edit any listing in the system from within the admin tool. This includes anything you can change on the client plus the ability to extend, change/remove attached files/images, change price plan attachment for renewal charging....etc.
Export/Import Listings
Tools to export listings within csv or xml format allowing you to import listings into other software or into other sites. Bulk listing import tool to allow manual and automatic import of listings from bulk csv files like those supplied by car dealerships and other sources...
Admin Listing Expiration
This allows you the admin to reset the expiration of any listings in the system. This allows the listing to go through the listing expiration process at a time you determine so the listing is still within the system. If you need a listing removed from the system completely simply login as the admin on the client side to remove immmediately and irrevocably.
Client Side Listing Management
You as the admin can log into the client side to manage listings also. This allows for quick listing edits as well as quick listing removals. The listing removal on the client side completely removes the listing from the system. .
Allowed HTML Filter
This feature allows the admin to control what html can be used within the clients listing. You have a point and click admin tool to ban the use of some html to limit html page issues and html hacks from clients. You can add your own custom tags to ban if you like...
Badword Filter
Ban or exchange words inserted by clients with more appropriate text or simply remove. You control the words searched/removed and what they are replaced with. Can be used to remove competitor urls from inclusion within your site if you allow linking to begin with...

General Admin Controls

Splash Pages During Registration
This feature allows you to display specific html in a specific page within the registration process to welcome or inform your new registrants. This feature is set on a user group by user group basis so you can control which user groups see what if you choose to show them anything.
Manage Pricing by User Groups
A price plan is attached to each user group that allows you to bill users in that user group a specific way. Each user group can have it's own price plan (way of charging)....
Multi-Admin Users
Create as many admin users within the system as you want. Create admin groups with specific rights and add users to them. You can tailor access and abilities down to a page by page basis as well as view/edit abilities within those pages..
Splash Pages During Listing Placement
This feature allows you to display specific html in a specific page within the front of the place a listing process to inform your clients of sales or other site features. This feature is set on a user group by user group basis so you can control which user groups see what if you choose to show them anything.
Category Browsing Configurations
There are many features within the software that can be configured on a category by category basis including browsing display, questions and fields, classified ad and auction durations, template use, browsing filters and many more...
Admin Account Login Reset
This feature allows the admin user to reset their admin accounts login information if ever lost..
Turn Off Powered by Free
Though we would like you to leave we understand if you want to remove it. You can turn off the "Powered by Geodesic Solutions" link at the bottom of your site though we would welcome you leaving it. There is no extra charge to turn this off
IP Banning Feature
Ban irritating or offensive users by their ip address. Insert ip address blocks.to ban to possibly ban entire countries from accessing your site. Use wildcards within ip address blocks. ..
Unlimited Categories
You can create as many categories within your site as you like. You can create as many category levels as you like also. Each level will have access to all the category specific features of the software..
Group Specific Sponsored By Display
One of the many features of user groups is that you can group users by their affiliations to outside groups and organizations. You may have users that all belong to a specific company, belong to a specific community group and any number of extra site affiliations. You can leverage that affiliation onto your site by grouping users by that affiliation using user groups. And then use this feature to display specific text about the group or icons or logos that represent that group within any listings users in that group place on the site.
